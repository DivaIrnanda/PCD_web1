Tentang Sistem

Sistem Prediksi Segmentasi Banjir adalah aplikasi web berbasis deep learning yang dirancang untuk membantu identifikasi dan pemetaan area genangan banjir secara cepat dan akurat.

Banjir merupakan salah satu bencana alam yang paling sering terjadi di Indonesia. Pemetaan area banjir secara manual melalui survei lapangan membutuhkan waktu lama dan sulit dilakukan ketika banjir terjadi secara mendadak. Oleh karena itu, sistem ini dikembangkan untuk menyediakan solusi otomatis yang dapat memproses citra dan menghasilkan segmentasi area banjir dalam hitungan detik.

Cara Kerja Sistem

  1. Upload Citra: User mengunggah citra melalui interface web. File disimpan dengan nama unik untuk mencegah collision.
  2. Preprocessing: Citra di-resize ke 256Ã—256 piksel dan dinormalisasi ke range 0-1. Format distandarisasi untuk input model.
  3. Prediksi Model: Model U-Net melakukan forward pass dan menghasilkan probabilitas untuk setiap piksel apakah termasuk area banjir atau tidak.
  4. Thresholding: Probabilitas di-threshold pada nilai 0.5. Piksel dengan probabilitas > 0.5 diklasifikasikan sebagai banjir.
  5. Visualisasi: Sistem membuat 3 jenis visualisasi: citra asli, mask hitam-putih, dan overlay dengan transparansi merah.
  6. Kalkulasi Statistik: Menghitung persentase area banjir, jumlah piksel banjir, dan total piksel.
  7. Penyimpanan: Semua file (original, mask, overlay) disimpan ke disk dan metadata disimpan ke database SQLite.
  8. Response: Hasil dikirim ke browser dalam format base64 untuk display real-time tanpa reload.

Keamanan dan Privasi

Sistem ini dibangun dengan mempertimbangkan aspek keamanan dan privasi data pengguna:

Data Privacy

Setiap user hanya dapat mengakses data milik mereka sendiri. Data prediksi bersifat private dan tidak dapat diakses oleh user lain.

Secure Storage

File citra dan hasil prediksi disimpan dengan nama unik dan terenkripsi untuk mencegah akses tidak sah.

File Upload Security

Validasi format file, sanitize filename, dan batasan ukuran file maksimal untuk mencegah malicious upload.

Data Control

User memiliki kontrol penuh terhadap data mereka dan dapat menghapus prediksi kapan saja.

Batasan Sistem

Penting untuk Diketahui:
  • Model memiliki akurasi 90.69%, namun tidak sempurna. Hasil prediksi sebaiknya digunakan sebagai alat bantu dan memerlukan validasi dari expert atau observasi lapangan untuk keputusan kritis.
  • Kualitas prediksi sangat bergantung pada kualitas input citra. Gambar yang blur, terlalu gelap, atau memiliki banyak noise dapat menghasilkan prediksi yang kurang akurat.
  • Model dilatih dengan dataset tertentu. Performa terbaik pada kondisi yang mirip dengan data training. Kondisi ekstrim atau atipical mungkin menghasilkan hasil yang kurang optimal.
  • Ukuran file maksimal 16MB. Citra yang terlalu besar perlu dikompres terlebih dahulu.
  • Sistem memerlukan koneksi internet yang stabil untuk upload dan prediksi.
